Задание: 
Разработать функцию, которая определяет является ли натуральное число квадратом какого либо числа. 
Пользователь вводит 3 натуральных числа для вычисления функция должна быть реализована 3 способами:
1) Функция которое возвращает искомое значение;
2) Функция, которая возвращает скомое значение через указатель;
3) Функция, которая возвращает искомое значение через ссылку. 
3 версии функции должны иметь одинаковое имя т.
